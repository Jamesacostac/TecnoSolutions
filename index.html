Limpieza de Laptop

Básico
Ver pasos del tutorial
  1. Apaga el equipo y retira la batería si es posible.
  2. Usa aire comprimido para limpiar las rejillas de ventilación.
  3. Limpia la pantalla con un paño de microfibra seco.
  4. Usa alcohol isopropílico para las teclas (con cuidado).

Optimización de Celular

Software
Ver pasos del tutorial
  1. Borra el caché de las aplicaciones pesadas.
  2. Desinstala apps que no uses hace más de un mes.
  3. Reinicia el dispositivo al menos una vez por semana.
